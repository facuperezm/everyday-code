# Codewars Katas

Solutions to **286 katas** from [Codewars](https://www.codewars.com/), organized by difficulty level.

## Difficulty Breakdown

| Kyu | Difficulty | Solutions | Description |
|-----|------------|-----------|-------------|
| 8 kyu | Beginner | 159 | Fundamentals, basic syntax |
| 7 kyu | Easy | 87 | Simple algorithms, string/array manipulation |
| 6 kyu | Medium | 36 | Intermediate algorithms, problem decomposition |
| 5 kyu | Hard | 4 | Advanced algorithms, optimization |

## Notable Solutions

### 5 kyu (Hard)
- **Maximum Subarray Sum** — Kadane's Algorithm implementation
- **Moving Zeros to the End** — In-place array manipulation
- **Extract Domain** — URL parsing and regex
- **Calculating with Functions** — Functional programming patterns

### 6 kyu (Medium)
- **Two Sum** — Classic interview problem
- **Mexican Wave** — String transformation
- **Tribonacci Sequence** — Dynamic sequence generation
- **Duplicate Encoder** — Frequency counting with hash maps
- **Create Phone Number** — String formatting
- **Highest Scoring Word** — ASCII value calculations

### 7 kyu (Easy)
- **Square Every Digit** — Number manipulation
- **Vowel Count** — Character frequency
- **Isogram Detection** — Set operations
- **Descending Order** — Sorting algorithms
- **Complementary DNA** — String mapping
