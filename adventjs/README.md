## Challenge Themes

AdventJS challenges are Santa/Christmas themed and typically involve:

- Gift management and sorting
- Elf task scheduling
- Sleigh route optimization
- Toy factory problems
- Present wrapping algorithms

## Skills Practiced

- Array filtering and sorting
- String manipulation
- Object transformation
- Algorithmic problem-solving
- TypeScript (2025)

## About AdventJS

AdventJS is an annual event created by [@midudev](https://github.com/midudev) that runs during December. Each day presents a new Christmas-themed coding challenge to solve with JavaScript or TypeScript.
